###vim的使用

##1.打开与退出(重点)

#两个模式：
1. 命令模式
2. 插入模式

>>>建立文件：touch 文件名
>>>打开文件：vim 文件名（可以直接用，不用touch）

#步骤：
输入文字：按i，从命令模式进入到插入模式
退出：按esc，从插入模式进入到命令模式
输入：：wq，保存并退出
输入：：q!，不保存并退出

##2.移动光标

```
上：k
下：j
左：h
右：l
移动到行首：0
移动到行尾：$
移动到文件开头：gg
移动到文件结尾：G
移动到指定行：:行号
```

##3.编辑操作

```
删除：x
删除光标前的字符：退格键
删除光标后的字符：删除键
删除整行：dd（加n）
复制：yy（加n）
粘贴：p
撤销：u
重做：Ctrl+r
```

##4.搜索与替换

```
/关键词

#全文替换 
:%s/旧关键词/新关键词/g

#替换当前行：
:s/旧关键词/新关键词/g

#替换当前行到文件结尾：
:%s/旧关键词/新关键词/gc

#替换当前行到文件开头：
:%s/旧关键词/新关键词/gc
```

##5.可视化模式
```
v → 按字符选中

V → 按行选中

Ctrl+v → 按列块选中(y复制，d删除，p粘贴)
```

##6.实用技巧

```
set number → 显示行号

set nonumber → 取消行号

syntax on → 开启语法高亮

set tabstop=4 → 设置 Tab 宽度为 4

set expandtab → Tab 转为空格

```
