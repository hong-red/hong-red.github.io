###虚拟机的搭建

##1.准备工作

#Vmware的安装
 >>>[官网下载](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion)
 #Ubuntu(光盘下载)
 >>>[官网下载](https://ubuntu.com/download/desktop)

 提醒：内存需要7GB以上，且下载时间较长

 ##2.安装虚拟机软件
 1. 打开Vmware软件
 2. 点击“创建新的虚拟机”
 3. 选择“自定义（高级）”
 4. 点击“下一步”
 5. 选择“稍后安装操作系统”
 6. 点击“下一步”
 7. 选择“Linux”
 8. 选择“Ubuntu 64位”
 9. 输入虚拟机名称
 10. 选择虚拟机存储位置
 11. 输入虚拟机磁盘大小（不要太小）
 12. 点击“下一步”
 13. 点击“完成”

 ##3.启动虚拟机并安装系统

选择 Install Ubuntu，跟着流程选择就行
记住你设置的密码，后面每次开机都要用

###最常见的问题

##VMware Tools未安装

#如何发现：
打开终端，看两个系统是否能正常使用粘贴
若没有正常使用，说明没有安装 VMware Tools

#原因：
1. 虚拟机未安装 VMware Tools
2. 安装了 VMware Tools 但未启动

#解决方法：
1. 点击“虚拟机”菜单，选择“安装 VMware Tools”
2. 插入 VMware Tools 光盘（一般1和2可跳过，因为一般都为2）
3. 打开终端，输入以下命令安装 VMware Tools
```
sudo apt-get install open-vm-tools
```
4. 安装完成后，重启虚拟机


##网络问题
一般可以直接使用，若不能使用，需要手动配置网络

#解决方法：
>>>先看更新了没，更新好了还不能用后可试试以下方法
1. 点击“编辑”菜单，选择“虚拟网络编辑器”
2. 选择“VMnet8”，点击“NAT设置”
3. 输入网关IP地址，例如：127.0.0.7890
4. 输入子网掩码，例如：255.255.255.0
5. 点击“确定”
6. 重启虚拟机

当真不行时，重装虚拟机
